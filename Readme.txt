#Car_Sales
This project contains a web application with the following functionality:

-Display vehicle inventory of a mock car dealership

-View all or search through the inventory

-Allow users to create a simple account and login

-Bidding feature for cars that have been "on-lot" for over 120 days

-Purchase vehicle (no actual payment page) and remove from inventory

-Generate sales forms

-Allow employees to login and have access to sales forms
